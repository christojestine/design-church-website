## 🎨 Color Breakdown by Section

### Hero Section
- **Background**: Navy gradient (from-[#04142D] to-[#081A38])
- **Text**: Off-white ([#F8F5EE])
- **Accents**: Gold ([#D4AF37]) for buttons and highlights
- **Opacity variations**: 70-80% for secondary text

### Mission Section
- **Background**: Dark navy gradient
- **Image Frame**: Glass effect with gold border
- **Text**: Off-white with gold accents
- **Decorative Cross**: Semi-transparent gold

### Cards (Experience, Events, Gallery)
- **Base**: White with 5% opacity (glassmorphism)
- **Border**: Gold with 30% opacity
- **Hover**: White opacity increases to 10%
- **Text**: Off-white with 80% opacity

### Statistics Section
- **Numbers**: Gold gradient text
- **Labels**: Off-white
- **Divider Lines**: Gold gradient

## 📏 Typography Hierarchy

```
H1 - Hero Headline     : 5xl-7xl bold (64-80px)
H2 - Section Title     : 5xl-6xl bold (48-64px)
H3 - Card Title        : xl-2xl bold (20-24px)
H4 - Subtitle          : lg-xl bold (16-20px)
Body - Main Text       : base-lg regular (16-18px)
Small - Caption        : sm text (14px)
Tiny - Badge/Label     : xs text (12px)
```

---

## 🔄 Responsive Breakpoints

```
Mobile:  < 640px    (1 column, full width)
Tablet:  640-1024px (2 columns)
Desktop: > 1024px   (3-4 columns)
```

## ✨ Visual Effects Summary

| Effect | Where | Duration |
|--------|-------|----------|
| Parallax | Hero image | Continuous on scroll |
| Float | 3D Cross, Prayer Orb, Particles | 3-8s infinite |
| Fade-up | All sections on scroll | 0.6-0.8s |
| Hover Lift | Cards | 0.3s smooth |
| Glow Pulse | Background decorations | 4s infinite |
| Bounce | Play button | 1.5s pulse |
| Rotate | Icons | Slow continuous |
